#### #1. `snake_body_straight.png`

| 항목 | 스펙 |
|------|------|
| **크기** | 20×20px |
| **배경** | 투명 (PNG-24) |
| **메인 색상** | `#00aa00` (어두운 초록) |
| **하이라이트** | `#00cc00` (좌상단 2px 밝은 부분) |
| **섀도우** | `#008800` (우하단 2px 어두운 부분) |

**디자인 설명:**
- 정사각형 타일 형태의 뱀 몸통 한 칸
- **중앙에 둥근 직사각형** 형태 (모서리 2px 라운드)
- 전체 20px 중 **가장자리 2px는 여백**으로 비워둠 (세그먼트 간 자연스러운 연결)
- **비늘 패턴**: 중앙에 작은 U자 모양 (2×1px) 비늘 무늬 2~3개 배치
- **입체감**: 좌상단→우하단 방향 그라데이션 느낌 (3단계 색상)
- 수직/수평 모두 사용 가능하도록 **상하좌우 대칭** 디자인

**참고 이미지 느낌:** 클래식 Nokia 스네이크 게임의 몸통 + 현대적 픽셀아트 질감

| 항목 | 스펙 |
|------|------|
| **크기** | 20×20px |
| **배경** | 투명 |

| **기본 방향** | 좌→하 (ㄴ자 형태) |

**디자인 설명:**
- 뱀이 방향을 틀 때 사용하는 **코너 세그먼트**
- **기본 형태**: 좌측에서 들어와서 아래로 나가는 ㄴ자

- 코드에서 90°, 180°, 270° 회전시켜 4방향 모두 사용
- **비늘 패턴**: 코너 중앙에 1개만 배치
- 직선 몸통과 **연결부가 자연스럽게 이어지도록** 가장자리 형태 일치시킬 것

**주의:** 회전해도 어색하지 않게 중앙 기준 디자인

## 4. 스타일 가이드라인

### 4.1 픽셀 아트 규칙

1. **해상도**: 지정된 크기 **엄격히** 준수
2. **배경**: 특별 명시 없으면 **투명 배경** (PNG-24)
3. **색상 수**: 스프라이트당 **최대 16색** 권장
4. **안티앨리어싱**: **절대 사용 금지** (픽셀 경계 선명)
5. **아웃라인**: 1px 검정 아웃라인 **권장** (가독성)
6. **그라데이션**: 2~4단계 색상 전환 (디더링 X)

### 4.2 파일 포맷

- **포맷**: PNG-24 (투명도 지원)
- **색공간**: sRGB
- **압축**: 무손실
- **네이밍**: `snake_case.png` (소문자, 언더스코어)
- **DPI**: 72 (웹용)
